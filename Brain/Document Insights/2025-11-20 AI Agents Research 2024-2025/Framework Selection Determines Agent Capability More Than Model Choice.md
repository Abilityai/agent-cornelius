## Core Insight

The **agent framework** (LangGraph, AutoGen, CrewAI) determines success more than underlying **model selection** (GPT-5 vs. Claude vs. Gemini), suggesting **system architecture** trumps **raw intelligence** for production agents. Each framework encodes different organizational paradigms creating fundamentally different capability profiles.

---

## Context & Evidence

**Three Dominant Paradigms**:

**LangGraph** (LangChain Ecosystem):
- **Paradigm**: Graph-based workflow with nodes and edges
- **Strength**: Managing intricate, multi-step processes where sequence and flow critical
- **Architecture**: Sophisticated state tracking and process management
- **Best For**: Complex workflows, precise control, production-grade flows
- **Learning Curve**: Steeper than CrewAI
- **Metaphor**: Assembly line with defined stations and transitions

**AutoGen** (Microsoft):
- **Paradigm**: Conversational collaboration (workflows as conversations between agents)
- **Strength**: Enterprise-focused infrastructure, robust reliability and error handling
- **Architecture**: LLM-to-LLM collaboration with research-grade flexibility
- **Best For**: Enterprise R&D, experimental setups requiring flexibility, pushing agent boundaries
- **Learning Curve**: Most comprehensive for enterprise deployments
- **Metaphor**: Conference call with multiple participants collaborating

**CrewAI**:
- **Paradigm**: Role-based model inspired by real-world organizational structures
- **Strength**: Rapid prototyping and developer experience
- **Architecture**: AI teamwork with specific roles and responsibilities like human teams
- **Best For**: Getting developers up and running quickly, intuitive project structure
- **Learning Curve**: Fastest path from concept to working implementation
- **Metaphor**: Human team with defined roles (designer, coder, QA)

## Context & Evidence (Continued)

**Why Framework Matters More Than Model**:

**1. Organizational Logic Built-In**:
- Framework determines how agents coordinate
- Model provides intelligence, framework provides organization
- Organization > intelligence for complex tasks

**2. Constraints Enable Performance**:
- Graph structure (LangGraph) enforces workflow discipline
- Role assignments (CrewAI) prevent responsibility diffusion
- Conversation patterns (AutoGen) enable exploratory collaboration

**3. State Management**:
- How memory persists between steps
- How context flows between agents
- How failures recover

**4. Tool Integration**:
- How agents access external systems
- How results incorporate into reasoning
- How errors handle

**Critical Insight**: Same model (e.g., GPT-4) in LangGraph vs. CrewAI produces **fundamentally different behaviors** because framework shapes:
- What information available when
- What actions possible at each step
- How agents collaborate
- How failures recover

**Implication**: Optimizing model selection (GPT-5 vs. Claude) yields marginal gains. Optimizing framework selection yields step-function improvements.
