[[To change the habit we need to change the routine but keep the cue and reward]]

In order for us to notice the Cue we need to practice Mindfulness to learn to See and recognize things our mind wants us to do. Only then we will be able to intervene and break the loop. 

Mindfulness needs to become a habit. Only then we can break these loops.
