# Diagram Visual Style Guide

## Color Palette
- **Background Layers**: Soft pale red (#FFE5E5), light gray (#F5F5F5), white (#FFFFFF)
- **Accent Color**: Bright red (#FF0000) for strategic highlights and borders
- **Text Color**: Dark gray (#2D2D2D) for all text
- **Node Fill**: Semi-transparent white (#FFFFFF) with subtle shadows
- **Stroke Color**: Light gray (#CCCCCC) with bright red accents for emphasis

## Design Principles
- Clean, modern, tech-forward aesthetic
- High contrast for readability
- Rounded corners for softer appearance
- Minimal yet impactful design
- Professional appearance with negative space
- Layered depth using subtle shadows

## Visual Elements
- **Nodes**: Rounded rectangles with semi-transparent white fill
- **Borders**: Light gray default, bright red for key elements
- **Arrows**: Clean lines in dark gray
- **Groups/Clusters**: Light background tints (pale red, light gray)
- **Shadows**: Subtle for depth without clutter

## Application
- Use gradient-like color progression across diagram sections
- Apply bright red strategically to highlight core components
- Maintain high contrast between elements
- Keep backgrounds subtle to not compete with content
- Ensure professional, tech-forward aesthetic throughout
